# Dockerized Multi-Tier WordPress Stack

## Overview

This project demonstrates a multi-container WordPress application using Docker Compose.

Components:

- Nginx
- WordPress
- MariaDB
- phpMyAdmin
- Docker Network
- Docker Volumes
